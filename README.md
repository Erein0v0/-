# 电费管家 ⚡

专为大学宿舍设计的微信小程序，按每人当日在宿时长比例自动分摊电费。

---

## 功能

- **记账**：选择室友，滑动填写当日缺席时长，一键提交
- **结算**：输入当日总度数，自动按在宿比例计算每人应缴金额
- **月历**：已结算日期标绿点，支持月份切换，查看历史账单
- **月度汇总**：当月每人应缴合计，按金额降序排列
- **区间统计**：自由选择日期范围，查询总度数、总电费及各人分摊

## 分摊算法

```
个人电费 = 当日总电费 × (个人在宿时长 / 所有人在宿时长之和)
```

电费单价：1 元 = 1.48 度

### 2. 导入微信开发者工具

### 3. 修改 `app.js`

```javascript
App({
  onLaunch() {
    // 替换为你的云开发环境 ID（云开发控制台 → 设置 → 环境设置）
    wx.cloud.init({ env: '你的云环境ID', traceUser: true })

  globalData: {
    // 替换为真实室友姓名
    roommates: ['室友A', '室友B', '室友C', '室友D'],
  },
})
```

### 4. 创建数据库集合

在云开发控制台 → 数据库 → 新建集合，名称填 `daily_bills`，安全规则设置为：

```json
{ "read": true, "write": true }
```

### 5. 编译运行

点击开发者工具中的**编译**，即可在模拟器或真机调试。
